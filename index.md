# VMware Tanzu Helm Repository

![VMware Tanzu](assets/vmware-tanzu-logo.png)

Add VMware Tanzu repository to Helm repos:

```bash
helm repo add vmware-tanzu https://vmware-tanzu.github.io/helm-charts
```
